# Dropbox Principal Product Manager - Interactive Job Description

A modern, interactive webapp showcasing the Principal Product Manager role at Dropbox, designed to provide an engaging candidate experience.

## Features

- **Interactive Role Explorer**: Dynamic content switching between role categories
- **Skills Compatibility Assessment**: Real-time matching algorithm with visual feedback
- **Media Workflow Showcase**: Interactive timeline of collaboration processes
- **Culture Quiz**: Multi-step assessment of cultural fit
- **Application Tracker**: Visual progress tracking for candidates
- **Responsive Design**: Optimized for all devices

## Technology Stack

- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- **Animations**: Anime.js, Typed.js, Splitting.js
- **Physics**: Matter.js for interactive elements
- **Data Visualization**: ECharts.js
- **Hosting**: GitHub Pages

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/dropbox-job-description.git
   cd dropbox-job-description
   ```

2. Start a local server:
   ```bash
   python -m http.server 8000
   ```

3. Open your browser to `http://localhost:8000`

## Deployment

This site is configured for GitHub Pages deployment. Simply push to the `main` branch and GitHub Pages will automatically deploy your site.
